# Contributing

Thank you for your interest in this repository!

This repository contains companion code for *AI-Native Infrastructure* by Jubin Soni,
published by Manning Publications Co. Manning holds the copyright to all content in this repo.

## How to Contribute

**Pull requests are not accepted.** Because Manning Publications Co. owns the copyright
to this work, we cannot accept code contributions that would create ownership complications.

## Reporting Issues

If you find a bug, typo, or error in the code samples, please open a GitHub Issue with:

- The chapter and listing number (e.g., `ch02 / listing_2_3`)
- A description of the problem
- The expected vs. actual behavior (for code bugs)

Issues are the preferred — and only — channel for feedback on this repository.

## Questions About the Book

For questions about the book's content, visit the Manning forum for this title at:
https://www.manning.com
